### 上手指南

###### 开发前的配置要求

1. 装有[python](https://python.org/)和[pip](https://pypi.org/)

###### 安装步骤

1. 克隆本仓库。

   ```shell
   git clone https://github.com/YANGRENRUIYRR/PM.git
   cd PM
   ```

2. 运行。

   ```shell
   python manage.py runserver
   ```
   打开<http://127.0.0.1:8000>即可看到结果
